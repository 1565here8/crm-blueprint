## BOT FORGE — MCP BUILD & DEPLOY (BASE-STYLE)

Dagulai Bot Forge is the no-code path from **plain-English strategy** to **deployed auto-trading bots**, orchestrated through **MCP (Model Context Protocol)** tools rather than manual API wiring.

### Architecture (brain vs hands)

| Layer | Role | Launch posture |
|-------|------|----------------|
| **Brain** | Rules, signals, risk limits, impact briefs → bot logic | Analyst tier and above (simulation + paper) |
| **Hands** | Exchange execution (Binance, Kraken, IBKR, etc.) via sealed API keys | Strategist+ when live; always label **simulation vs live** |

### MCP workflow (target)

1. User describes bot intent in Bot Forge UI or chat.
2. MCP server exposes tools: list_strategies, alidate_bot_spec, deploy_bot, ot_status, pause_bot, 
otate_keys.
3. Agent composes a normalized bot spec (conditions, symbols, max drawdown, exchange connector).
4. Deploy returns a **RUNNING** / **PAPER** / **FAILED** state with audit log — same mental model as Base MCP deploy flows.
5. Pulse Desk + impact briefs feed the brain; hands never run without explicit user consent and tier gate.
